# Paper Figures

Guidelines for figure generation and management.

## Figure Naming Convention

```
fig_{section}_{number}_{description}.{format}
```

Example: `fig_05_01_agent_architecture.svg`

## Recommended Formats

- **SVG**: Vector diagrams (architecture, workflows)
- **PNG**: Raster images (screenshots, photos)
- **PDF**: Publication-ready composite figures
